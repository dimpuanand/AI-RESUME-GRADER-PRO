# 📖 Overview

AI Resume Grader Pro is an intelligent web application that helps students, fresh graduates, and job seekers improve their resumes using Artificial Intelligence.

The application analyzes uploaded resumes, compares them against target job roles, calculates an ATS (Applicant Tracking System) score, identifies missing skills, and generates personalized recommendations using Google Gemini AI.

The platform also includes resume comparison, AI enhancement tools, interactive career roadmaps, professional PDF report generation, evaluation history, and a fully responsive interface for desktop, tablet, and mobile devices.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/dimpuanand/AI-RESUME-GRADER-PRO.git
```

Go to the project directory

```bash
cd AI-RESUME-GRADER-PRO
```

Install dependencies

```bash
npm install
```

Start the application

```bash
npm run dev
```

Build the project

```bash
npm run build
```
